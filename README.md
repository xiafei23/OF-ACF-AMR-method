代码准备了两套版本，分别是包含简单demo的测试模式与核心代码模式，包含demo的测试文件可以直接运行得到结果，核心代码需要您自己配置环境、接入数据集并根据论文完成参数配置。
Two versions of the code are provided: a test mode that includes a simple demo, and a core code mode. The test files with the demo can be run directly to produce results, while the core code requires you to configure the environment, integrate the dataset, and configure the parameters according to the paper.
